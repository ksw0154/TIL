# JavaScript [선형 자료구조3]

## 스택

- 나중에 넣은 데이터가 먼저 나오는 LIFO(Last In First Out) 기반의 선형 자료 구조

### 구현 메소드

- 데이터 전체 획득 / 비어있는지 확인 : getBuffer(), isEmpty()
- 추가 / 삭제 / 마지막 데이터 조회 / 크기 확인 : push(), pop(), peak(), size()
- 데이터 위치 / 존재 여부 확인 : indexOf(), includes()

- 사용 예시 : Ctrl + z, 뒤로 가기(페이지 이동)
